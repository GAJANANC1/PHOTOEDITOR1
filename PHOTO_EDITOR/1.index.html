FEATURES

SET MEN STYLES

In men styles there are four option.with the help of first option we can add hairstyle to images.second feature is adding mustache.we can add it to images and with the help of resize button we can fit it to image as required .Third option is add beard.there are many kinds of beard.you can select any of these.Fourth option is to add eyebrow. with its help we can set eyebrow of a person

SET ACCESSORIES

Set accessories has four options. 1.Sunglasses:- with the help of this option we can add sunglasses to images. 2.Cap:-with the help of these option we can add cap to the image 3.chain:this option is used to add chain 4.tie:this option is used to add tie

SET EFFECTS

This options contains three feature Adjust,Color and filter.with the help of adjust we can set contrast,brightness. color container helps to change the color of images. filter has many shads such grayish,bluish and many more.

PICTURE EDIT

It containes features such as flip horizontal,flip vertical,rotate left,rotate right,resize,greyscale. flip horizontal flips the image horizontally.flip vertical flips the image vertically.rotate left rotate the image towards left side.resize resizes the image according to given height and width.greyscale converts the image into black and white.
